# 今天学习了使用中间件和 EJS 模板

## 1. 创建新项目, 初始化 package.json, 并下载依赖

```bash
# 初始化 package.json, -y 参数是执行快速初始化
sudo npm init -y
```

```bash
# 下载 express 模块
sudo npm install express --save
# 下载 ejs 模块
sudo npm install ejs --save
```

## 2. 根目录创建 app.js 并引入 express, ejs, fs 模块, 实例化服务器对象

```js
// app.js

let express = require("express");
let ejs = require("ejs");
let fs = require("fs");
let app = express();
```
